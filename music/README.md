# Happy Soaring Music — ficheiros de áudio

Coloca aqui os MP3 das faixas (ex.: soaring-golden-hour.mp3).
Recomendado: pré-escutas curtas (30–60s), ~128 kbps, para proteger o trabalho
e o site carregar depressa. A faixa completa entregas ao cliente após o contacto.

Os caminhos usados no site apontam para music/<ficheiro>.mp3 e são editáveis no CMS
(cada faixa tem o campo "Ficheiro MP3").
